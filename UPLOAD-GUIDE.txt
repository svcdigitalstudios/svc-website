UPLOAD TO GITHUB:
1. Extract this ZIP.
2. Open the GitHub repository: svc-website
3. Click Add file > Upload files.
4. Upload ALL files from the extracted folder.
5. Keep every file in the repository root.
6. Select “Commit directly to the main branch”.
7. Click Commit changes.
8. Wait 2–3 minutes and refresh the website.

Do not upload only index.html. The gallery, temple and moment WebP files are also required.
